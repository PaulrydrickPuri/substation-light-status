# Substation Light Panel Status Detection

> YOLO-based real-time detector for electrical switchgear panel indicator light states (ON/OFF) using a fixed CCTV camera.

---

## Project Overview

This project builds an object detection model to monitor the status of indicator lights on an ABB electrical switchgear cabinet viewed from a fixed CCTV camera. The model detects each light's colour (red, green, yellow, blue) and state (ON/OFF) in real-time, enabling automated substation health monitoring without manual inspection.

## Recommended Training Config

```
Architecture : yolov8m  |  Fine-Tune : Yes
Input        : 640      |  Batch     : 8
Base LR      : 0.0001   |  Epochs    : 250   |  Patience : 40
Scheduler    : WarmupCosineLR
Optimizer    : SGD + Nesterov + Weight Decay 0.0001 + Gradient Clipping

Augmentation (key):
  Brightness 0.7–1.4 55% | Contrast 0.7–1.3 55% | Blur 50%
  Perspective 0.05–0.12 45% | Shadow 50% | Defocus 35%
  HSV Hue ±3° only (colour is class signal — do not shift more)
  NO Flip L/R | NO Rain/Fog/Snow (indoor fixed camera)
```
